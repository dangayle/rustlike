# FizzBuzz

Pattern matching and control flow.

## Run

```bash
pnpm fizzbuzz           # Runs index.ts (rust-like)
pnpm tsx 03-fizzbuzz/idiomatic.ts  # Runs idiomatic version
```

## What it demonstrates

- Using `Result` for validation
- Discriminated unions for representing different outcomes
- Pattern matching with `match()`
- Error handling in loops
- Transforming results with `map()`
