# Hello World

Basic `Option` usage with simple greetings.

## Run

```bash
pnpm hello           # Runs index.ts (rust-like)
pnpm tsx 01-hello-world/idiomatic.ts  # Runs idiomatic version
```

## What it demonstrates

- Creating `Some` and `None` values
- Using `unwrapOr()` for defaults
- Chaining with `map()`
- Pattern matching with `match()`
