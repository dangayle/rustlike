# Iris Classification

k-NN classifier with validation.

## Run

```bash
pnpm iris           # Runs index.ts (rust-like)
pnpm tsx 06-iris-classification/idiomatic.ts  # Runs idiomatic version
```

## What it demonstrates

- Input validation with `Result`
- Chaining validations with `andThen()`
- Data processing pipeline with error handling
- Using `Result` for ML predictions
- Transforming and extracting data with `map()`
- Comprehensive error messages for invalid inputs
